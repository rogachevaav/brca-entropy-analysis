# Results

This directory contains processed tables generated by the analysis notebook.

## Main summary tables

- `entropy_summary.csv` — descriptive statistics for exon and intron entropy
  by gene and species.
- `length_entropy_correlations.csv` — Spearman correlations between region
  length and entropy, calculated separately for exons and introns.
- `region_entropy_extremes.csv` — regions with minimum and maximum entropy.
- `sliding_window_summary.csv` — summary statistics for local entropy profiles.
- `sliding_window_extremes.csv` — coordinates of minimum and maximum profile
  windows.

## Detailed output

- `sliding_window_profiles.csv` contains all individual sliding-window
  calculations for all analyzed loci.

Because this table contains several thousand rows, GitHub may display it as
plain text instead of an interactive CSV preview. The file can still be
downloaded or loaded directly with pandas.
